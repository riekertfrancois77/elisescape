# Module 7 — Create Worlds

**Status:** OPENED by Dad (2026-07-07) · **Unlocks when Module 6 is complete.**
**Role:** Eli is Creative Director **and Art Director**. Claude wears the
**Technical Artist** hat — Eli directs the look; Claude paints it.

**Version target:** the game climbs from **0.3 → 0.4** — the biggest visual
glow-up yet.

## Purpose

Until now the game has *worked*; now it gets a **face**. This is the look-and-feel
module: real graphics, lighting, atmosphere, immersion. The whole point is that
**Eli art-directs** — the mood, the palette, the feeling all come from his head
first — and the studio paints to his direction. He learns that a game's look is a
*feeling* you decide on purpose.

## The Brain-First Rule

**Eli decides the look before any pixels exist.** Do not open by generating art.
Open by asking him what it should *feel* like — references, colors, mood, time of
day, the emotion he wants a first-time player to have. Only once he's art-directed
does Claude paint. Then he reacts and redirects — that loop is the module.

## What You'll Learn

- A game's look is a **feeling** — you decide the feeling first.
- **You art-direct** (references, mood, palette); the studio paints.
- Real graphics and atmosphere are the biggest glow-up the game gets.
- **Immersion is layered:** light + motion + sound together.
- Your taste leads every visual call — the AI serves it.

## Rules for Claude Code in This Module

- Start with **Eli's art direction in his own words** — capture it (mood board in
  text, palette, references) *before* building anything.
- Stay inside the studio's constraints: **HTML / CSS / JavaScript only, no
  frameworks, no external image files or libraries** — atmosphere is built with
  CSS (gradients, shadows, lighting), canvas, and code, the same way the foyer
  already is. (Real illustration/asset pipelines can come later; this module is
  about doing a lot with the tools in hand.)
- Build to his direction, then **show, react, redirect** — several passes. His
  taste decides when it's right; he's allowed to reject a pass.
- **Layer immersion:** ambient motion (flicker, drift), lighting that reacts,
  and a sound layer that matches the new mood.
- Keep the game **playable** — the puzzle and deduction chain must still work and
  still be readable through the new art. Verify it in a browser.
- **Bump the version to 0.4** in `MEMORY.md`; add a Latest Build Review pass
  (Art Director lens especially) and Session-Memory + `session-log` entries in
  Eli's voice.
- Update Module 7 Steps; unlock **World Builder** at the end.

## Mission Flow

1. **Meet the Technical Artist** — you art-direct, the studio paints.
2. **Art-direct first** — describe the look and mood you want, in your own words.
3. **Lock a palette + atmosphere** — set the vibe before a single pixel.
4. **The studio paints** — Claude builds real graphics/atmosphere to your direction.
5. **React & redirect** — what's right, what's not-you; send it back.
6. **Add immersion** — layers of light, ambient motion, and sound.
7. **Version bump** — your game has a real face now: **Version 0.4**.
8. **Achievement unlock:** World Builder.

## Done When

- Eli art-directed the look **before** any art was made.
- The game has a visibly upgraded look and atmosphere he's happy with.
- It's still playable and the puzzle still reads clearly.
- The game is bumped to **Version 0.4** and the memory is updated.
- The **World Builder** achievement is unlocked.
