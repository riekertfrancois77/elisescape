# Module 4 — Build a Second Brain

**Status:** OPENED by Dad (2026-07-06) · **Unlocks when Module 3 is complete.**
**Role:** Eli is Creative Director. Claude is the studio's build team.

## Purpose

Eli has already *felt* the studio remember his game between sessions — every
time he refreshed the dashboard and his progress was still there. Now he learns
the trick behind it: **project memory**. He opens the memory files, sees what
they hold, and starts writing to them himself — so every session starts smarter
than the last.

This module makes the invisible visible: the thing that's been quietly powering
the whole studio becomes something Eli understands and owns.

## What You'll Learn

- AI **forgets** between sessions — the *project* has to remember for it.
- The memory files (`MEMORY.md`, `logs/`) are the studio's **shared brain**.
- Good builders leave **notes for their future selves**.
- Writing memory **in your own words** makes it truly yours.
- Memory is what turns a *chat* into a *studio*.

## Rules for Claude Code in This Module

- Show, don't lecture. Open the real files with Eli and walk through what each
  one holds (`MEMORY.md` = current state; `logs/session-log.md` = the story so
  far; `logs/decisions-made.md` = locked calls).
- Ask Eli to summarize his game and where he's at **in his own words**, then
  help him write that into `MEMORY.md`. His words, not yours.
- Prove it works: end the session, then (next time) start cold and show how you
  pick up exactly where he left off *because the project remembered*.
- Update `MEMORY.md` and `logs/session-log.md` as always.

## Mission Flow

1. **Open the memory** — read `MEMORY.md` and the logs; see what the studio
   remembers about the game.
2. **Learn the files** — what each memory file is for.
3. **Summarize in your own words** — what your game is, where you're at.
4. **Write it into memory** with Claude.
5. **Test it** — end the session, start fresh, watch Claude continue seamlessly.
6. **Achievement unlock:** Memory Keeper.

## Done When

- Eli has read and understood the studio's memory files.
- He's written a summary of his game **in his own words** into `MEMORY.md`.
- He's seen the memory work across a fresh session.
- The **Memory Keeper** achievement is unlocked.
